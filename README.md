# Smart Resource Allocations

## Unique Feature Added: **Moonshot Slot**

This project now includes a unique feature designed for hackathons and product teams:

- A configurable fraction of total capacity is automatically reserved as a **Moonshot Slot**.
- That reserved capacity is awarded to the most promising **historically underfunded moonshot request**.
- This avoids local maxima and keeps long-shot, high-upside bets alive.

In short: the allocator balances fairness for core work **and** protected exploration for bold ideas.
